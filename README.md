# 软件工程： 实验一
## 实验目标及要求：
目标：
1. 熟悉面向对象编程；
2. 掌握利用大模型辅助编程的方式；
3. 实验同大模型的“结对编程’’。
4. 熟练掌握Git的基本指令和分支管理指令；
5. 掌握Git支持软件配置管理的核心机理；
6. 在实践项目中使用Git /Github管理自己的项目源代码。
要求：开发一个程序，实现从文本文件中读取数据并根据要求生成图结构，输出该图结构，并在其上进行一系列计算操作，实时展示各操作的结果。
